# 08_Список нерешённых вопросов

## Назначение

Этот файл хранит только вопросы, ответы на которые нужны для дальнейшей правки файлов `_Project Core/Navigation/`.

Если вопрос не влияет на `00–09`, он не должен оставаться здесь. Его нужно вынести в отдельный рабочий backlog по будущему лору или в профильную лорную заметку, когда такая заметка появится.

`08` не является анкетой по всему миру. Это рабочий список неопределённостей, мешающих довести Navigation до точного и аккуратного состояния.

## Правила оформления

- Каждый вопрос оформляется отдельным пунктом списка.
- Под каждым вопросом обязательно оставляется пустое поле для ответа автора:
	- `Ответ:`
- При добавлении нового вопроса в этот файл ассистент должен автоматически добавлять под ним поле `Ответ:`.
- Если автор ответил на вопрос, ассистент должен перенести ответ в соответствующий Navigation-файл, а не копить решённые ответы здесь.
- После переноса ответа вопрос удаляется из `08` или переформулируется, если осталась более узкая неопределённость.
- Если вопрос оказался связан не с Navigation, а с будущей лорной статьёй, его нужно вынести из `08`.

## Как отвечать автору

Можно отвечать прямо в этом файле под вопросом:

```md
- Вопрос?
	- Ответ: ответ автора
```

Если ответ не окончательный, это лучше явно пометить в самом ответе: `не канон`, `рабочая версия`, `старая идея`, `отложить`, `не использовать`, `не уверен`.
