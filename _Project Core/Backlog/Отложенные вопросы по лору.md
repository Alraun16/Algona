# Отложенные вопросы по лору

## Назначение

Этот файл хранит вопросы, вынесенные из `08_Список нерешённых вопросов`, потому что они не влияют напрямую на текущие файлы Navigation.

Их не нужно разбирать перед доведением Navigation до рабочего состояния. Они пригодятся позже, когда начнётся перенос подробного лора в профильные заметки.

Ответы на эти вопросы не должны автоматически становиться каноном. Если вопрос раскрывается позже, итоговое решение нужно переносить в соответствующую лорную заметку или системный файл, а не копить здесь.


## Вынесенные служебные вопросы из старой версии `08`

Эти пункты убраны из `08`, потому что они не являются текущими нерешёнными вопросами для правки Navigation. Часть из них уже решена автором, часть относится к будущей организации лорных заметок.

- Нужно ли перестраивать папки под явные разделы: `География`, `Расы`, `Государства`, `История`, `Магия`, `Религии`, `Персонажи`, `Бестиарий`, или оставить более органичную структуру по мирам и регионам?
- Верхнеуровневые папки `Аран'алас`, `Вейл'арис`, `Вирс'ахт` должны оставаться рядом с `Альгона` или быть вложены в общий раздел миров?
- Что делать с пустым файлом [[Два материка]]: превратить в обзор, удалить, переименовать или оставить заготовкой?
- Использовать ли содержимое `.trash` как возможный источник старого лора или полностью игнорировать?
