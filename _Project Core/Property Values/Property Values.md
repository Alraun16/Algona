# Field Enums

## type
- `races` — Расы
- `nation` — Нации
- `character` — Персонажи
- `species` — Существа в бестиарии, общие виды
- `region` — Области, земли
- `settlement` — Поселения
- `state` — Государство, королевство, империя, княжество
- `organization` — Гильдии, ордена, школы, учреждения, формальные структуры
- `religion` — Религии, культы, верования
- `event` — События, войны, реформы, основания, катастрофы, расколы

## status
- `idea` — Заметка-заготовка
- `draft` — Черновик
- `in_progress` — В активной доработке
- `review` — Требует проверки
- `final` — Рабочий утверждённый канон
- `archived` — Устаревший черновик

## canon_level
- `core` — Базовые истины мира
- `major` — Важный подтверждённый канон
- `normal` — Обычный рабочий канон
- `minor` — Малозначимая, но каноничная деталь
- `tentative` — Предположительный канон
- `alternate` — Альтернативный канон
- `deprecated` — Больше не действующий канон
